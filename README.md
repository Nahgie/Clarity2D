# Clarity2D

**Clarity2D** 는 기존 2D 게임 엔진인 **ClarityEngine** 을 리팩터링·개선하기 위해 새로 만든 실험적인 버전입니다.  
레거시 코드를 그대로 가져오는 대신, **모던 C++ 스타일**과 **C++26 프리뷰 기능**을 적극적으로 활용해 설계·구현하고 있습니다.

> ⚠️ **실험용 프로젝트입니다.**  
> API와 내부 구조는 언제든지 변경될 수 있으며, 실서비스/상용 프로젝트에 바로 사용하는 것은 권장하지 않습니다.

---

## 목표

- 기존 **ClarityEngine** 구조를 단순화하고 유지보수성을 높이기
- **모던 C++ (C++20 이상)** 문법과 패턴을 엔진 전반에 적용
- **C++26 프리뷰 기능**을 실제 엔진 코드에 시험 적용
- 순수 Win32 + 정적 라이브러리 기반으로 가벼운 2D 게임 엔진 지향

---

## 개발 환경

- **IDE**: Visual Studio 2026 (VS2026)
- **컴파일러**: MSVC  
- **플랫폼 도구 집합**: `v145`
- **C++ 표준**:  
  - 기본: C++20 이상  
  - 일부 모듈/컴포넌트: **C++26 프리뷰(/std:c++latest)** 사용

---

## 빌드 상태

> 아직 초기 개발 단계입니다.  
> 프로젝트 구조와 빌드 스크립트, 모듈 구성 등이 수시로 변경될 수 있습니다.
